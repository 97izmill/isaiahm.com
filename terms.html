The Program is a personal automated notification tool that the operator built for the operator's own use. It monitors public information sources on the internet and sends SMS notifications to the operator's own mobile phone when monitored conditions are met. 3. Single Sender and Single Recipient

The operator is the sole sender and the sole recipient of messages from the Program. The operator's mobile phone number is the only number that has been added to the Program's local configuration file. The Program has no public sign-up flow because no other person is intended to receive messages.

A record of the Program's opt-in (a structural excerpt of the configuration file with the personal number redacted) is published at https://isaiahm.com/sms-opt-in.html for verification by mobile carriers and The Campaign Registry.
