Program Name

"Inventory and Health Alerts" — an automated notification application operated by Isaiah Miller, currently in private operation with a single subscriber.

Program Description

The application is an automated script that runs on the operator's server. It monitors public information from public retailer websites and sends automated SMS notifications to subscribers when a vehicle matching their saved search criteria appears, and when the application's monitoring routines fail. All messages are application-generated; none are manually composed.

How a Subscriber Opts In

A subscriber opts in by being added to the "subscribers" list in the application's configuration file. Each addition goes through an explicit request → approve/deny step using the application's subscription tool (subscribe.py) on the operator's server: an opt-in request records the subscriber's name, phone number, the full consent statement, and a timestamp; the operator must then approve the request before the number is written into the configuration file, or deny it, in which case the number is never added. Every request, approval, and denial is recorded in an audit log. The current sole subscriber is the operator (Isaiah Miller), who self-subscribed on May 25, 2026 by adding their mobile phone number to the configuration file. Adding a number to the subscribers list is a deliberate, affirmative act that constitutes express written consent to receive automated SMS messages from the application.

A subscriber may unsubscribe at any time by replying STOP to any message from the application, or by being removed from the subscribers list in the configuration file. Replying HELP will return the operator's contact email.
