Program Name

CarMax Inventory Monitor — a personal SMS notification service.

Program Description

This service sends SMS notifications to the sole proprietor’s own mobile phone when a new vehicle matching the proprietor’s personal search criteria is listed on a public used-car retailer’s website. The sender and the sole recipient are the same individual.

Message Frequency

Recurring messages. Frequency varies based on retailer inventory changes. Typical volume is fewer than 10 messages per week.
